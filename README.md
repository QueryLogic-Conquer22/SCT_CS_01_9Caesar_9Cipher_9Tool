# 🔐 Caesar Cipher Encryption & Decryption Tool

## 📌 Project Overview

The Caesar Cipher Encryption & Decryption Tool is a Python-based application developed as part of my Cyber Security Internship at SkillCraft Technology.

This project demonstrates the implementation of the Caesar Cipher, one of the earliest and most widely known encryption techniques. The tool allows users to encrypt and decrypt messages using a custom shift value while preserving spaces, numbers, and special characters.

---

## 🚀 Features

* Encrypt plain text messages
* Decrypt encrypted messages
* User-defined shift values
* Supports uppercase and lowercase letters
* Preserves spaces and special characters
* Menu-driven interface
* Beginner-friendly cryptography project

## 🔒 How Caesar Cipher Works

The Caesar Cipher works by shifting each alphabet letter by a fixed number of positions.

Example:

Original Message:
HELLO

Shift Value:
3

Encrypted Message:
KHOOR

Decryption reverses the process and retrieves the original message.
